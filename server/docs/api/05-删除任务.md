# 删除任务

## 接口信息

- **URL**: `/api/tasks/{id}`
- **方法**: `DELETE`
- **描述**: 删除指定任务（物理删除，不可恢复）

## 请求参数

### Path参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | Long | 是 | 任务ID |

## 请求示例

```bash
curl -X DELETE http://localhost:8080/api/tasks/1
```

```javascript
fetch('http://localhost:8080/api/tasks/1', {
  method: 'DELETE'
})
.then(response => response.json())
.then(data => console.log(data));
```

## 响应示例

### 成功响应

```json
{
  "code": 200,
  "message": "操作成功",
  "data": null
}
```

### 失败响应（任务不存在）

```json
{
  "code": 500,
  "message": "任务不存在"
}
```

## 响应字段说明

成功时data字段为null。

## 业务规则

1. 任务ID必须存在，否则返回错误
2. 删除操作是物理删除，数据永久删除，不可恢复
3. 删除后任务立即从列表中移除
4. 删除操作不可撤销，前端应提供二次确认

## 注意事项

1. 删除操作不可恢复，请谨慎操作
2. 建议前端在删除前弹出确认对话框
3. 删除成功后，任务ID将不再可用
4. 删除操作会立即生效，无需刷新列表

## 前端建议

```javascript
// 删除前确认
function deleteTask(id) {
  if (confirm('确定删除该任务？删除后不可恢复')) {
    fetch(`http://localhost:8080/api/tasks/${id}`, {
      method: 'DELETE'
    })
    .then(response => response.json())
    .then(data => {
      if (data.code === 200) {
        // 删除成功，刷新列表
        loadTaskList();
      } else {
        alert(data.message);
      }
    });
  }
}
```

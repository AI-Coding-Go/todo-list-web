## 📖 文档说明

### API接口文档

- **位置**: `docs/api/`
- **说明**: 每个API接口都有独立的Markdown文档，包含详细的接口说明、请求参数、响应格式、示例代码等
- **索引**: 查看 `docs/api/README.md` 获取所有接口的快速导航

### 项目开发文档

- **位置**: `project/` 目录
- **说明**: 包含项目开发所需的所有文档和脚本
  - 需求文档
  - 开发方案
  - 开发指南
  - 数据库初始化脚本

### 项目根目录文档

项目根目录下的文档：

- **README.md**: 项目README，包含快速开始指南
- **快速启动指南.md**: 详细的启动说明
- **文档索引.md**: 文档导航
- **测试结果总结.md**: 功能测试结果

## 🚀 快速开始

### 前端开发人员

1. 查看 `docs/api/README.md` 了解所有可用接口
2. 根据需求查看对应的接口文档（如 `01-新增任务.md`）
3. 参考文档中的请求示例和响应格式进行开发
4. 使用 Swagger UI 进行接口测试：http://localhost:8080/swagger-ui.html

### 后端开发人员

1. 查看 `project/后端开发方案-完整版.md` 了解整体架构
2. 查看 `project/需求文档.md` 了解业务需求
3. 参考代码实现和单元测试

### 其他团队（开发产品）

1. 查看 `project/README.md` 了解项目文档
2. 按照 `project/开发指南.md` 进行开发
3. 参考 `project/` 目录下的所有文档

本目录包含项目的完整需求和开发方案文档，供其他团队参考和开发使用。

## 📋 文档列表

### 1. 需求文档
- **需求文档.md** - 完整的产品需求文档
  - 项目概况
  - 核心功能需求
  - 功能拆分
  - 预期成果
  - 补充说明

### 2. 开发方案
- **后端开发方案-完整版.md** - 完整的后端开发方案
  - 项目概述和技术架构
  - 数据模型设计
  - 系统架构设计
  - API接口设计
  - 核心功能实现
  - 技术实现细节
  - 开发环境配置
  - 测试和部署说明

### 3. 数据库脚本
- **schema.sql** - 数据库初始化脚本
  - 创建数据库和表结构
  - 包含索引设计
  - 执行前会清空已存在的数据库

## 🎯 使用说明

### 对于需要开发产品的团队

1. **第一步：了解需求**
   - 阅读 `需求文档.md` 了解产品功能需求
   - 理解业务逻辑和功能点

2. **第二步：了解技术方案**
   - 阅读 `后端开发方案-完整版.md` 了解技术架构
   - 查看数据模型设计
   - 了解API接口设计

3. **第三步：查看代码实现**
   - 参考项目源代码（`src/` 目录）
   - 查看API接口文档（`docs/api/` 目录）

4. **第四步：环境搭建**
   - 按照开发方案中的环境配置说明搭建开发环境
   - 执行数据库初始化脚本（`schema.sql`）
   - 启动服务进行测试
   
**详细步骤**: 请参考 `开发指南.md` 获取完整的开发步骤说明

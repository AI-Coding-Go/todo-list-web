## 🎯 按角色查找文档

   - `docs/api/README.md` - API接口索引
   - 需要使用的具体接口文档（如 `docs/api/01-新增任务.md`）

2. **参考**:
   - Swagger UI: http://localhost:8080/swagger-ui.html
   - `docs/API文档说明.md` - API文档使用说明

### 后端开发人员

1. **必读**:
   - `后端开发方案-完整版.md` - 完整开发方案

2. **参考**:
   - 代码实现（`src/` 目录）
   - 单元测试（`src/test/` 目录）

   - `docs/project/README.md` - 项目文档说明
   - `docs/project/开发指南.md` - 完整的产品开发指南
   - `docs/project/需求文档.md` - 产品需求文档
   - `docs/project/后端开发方案-完整版.md` - 完整的技术方案

2. **参考**:

   - `测试结果总结.md` - 了解功能实现情况

```
├── 测试结果总结.md                 # 测试结果

    ├── README.md                   # 文档目录说明

    ├── project/                    # ⭐ 项目文档目录（开发专用）
    │   ├── README.md               # 项目文档说明
    │   ├── 开发指南.md             # 完整开发指南
    │   ├── 需求文档.md             # 产品需求文档

```

# 待办清单 H5 应用运行指南

## 安装步骤

1. 进入项目目录
   ```bash
   cd client-react-plugin-glm
   ```

2. 安装依赖
   ```bash
   pnpm install
   ```

3. 启动开发服务器
   ```bash
   pnpm dev
   ```

4. 打开浏览器访问 http://localhost:3000

## 构建生产版本

```bash
pnpm build
```

构建完成后，文件将输出到 `dist` 目录。

## 功能说明

### 主要功能

1. **任务管理**
   - 添加新任务（点击右下角"+"按钮）
   - 编辑任务（点击任务右侧菜单）
   - 标记完成/未完成（点击复选框）
   - 删除任务（点击任务右侧菜单）

2. **筛选与排序**
   - 筛选：全部/未完成/已完成
   - 排序：最新添加/即将截止/优先级

3. **数据统计**
   - 完成率统计
   - 任务趋势图
   - 状态分布图

4. **提醒功能**
   - 截止前30分钟提醒
   - 截止时间提醒
   - 逾期提醒

### 优先级设置

任务支持三种优先级：
- 高（红色标记）
- 中（黄色标记）
- 低（绿色标记）

### 移动端适配

- 专为移动设备优化
- 支持触摸操作
- 响应式布局
- 安全区域适配

## 注意事项

1. 数据存储在浏览器的 localStorage 中，清除浏览器数据会导致任务丢失
2. 提醒功能需要页面保持活跃状态
3. 建议使用现代浏览器以获得最佳体验

## 问题排查

如果遇到问题，请尝试：

1. 清除浏览器缓存
2. 使用最新版本的现代浏览器
3. 确保设备有足够的存储空间
4. 检查浏览器控制台是否有错误信息

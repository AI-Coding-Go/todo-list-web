## 标记任务未完成

| 项目 | 内容 |
| :--- | :--- |
| **接口地址** | `/api/tasks/{id}/pending` |
| **请求方式** | `PATCH` |
| **请求数据类型** | `application/x-www-form-urlencoded` |
| **响应数据类型** | `*/*` |
| **接口描述** | 将已完成任务重新标记为未完成 |

### 请求参数

| 参数名称 | 参数说明 | 请求类型 | 是否必须 | 数据类型 | schema |
| :--- | :--- | :--- | :--- | :--- | :--- |
| id | 任务ID | path | true | integer(int64) | - |

### 响应状态

| 状态码 | 说明 | schema |
| :--- | :--- | :--- |
| 200 | OK | ApiResponseTodoTaskResponse |

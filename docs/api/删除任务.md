## 删除任务

| 项目 | 内容 |
| :--- | :--- |
| **接口地址** | `/api/tasks/{id}` |
| **请求方式** | `DELETE` |
| **请求数据类型** | `application/x-www-form-urlencoded` |
| **响应数据类型** | `*/*` |
| **接口描述** | 删除指定的任务 |

### 请求参数

| 参数名称 | 参数说明 | 请求类型 | 是否必须 | 数据类型 | schema |
| :--- | :--- | :--- | :--- | :--- | :--- |
| id | 任务ID | path | true | integer(int64) | - |

### 响应状态

| 状态码 | 说明 | schema |
| :--- | :--- | :--- |
| 200 | OK | ApiResponseVoid |

### 响应参数

| 参数名称 | 参数说明 | 类型 | schema |
| :--- | :--- | :--- | :--- |
| success | - | boolean | - |
| data | - | object | - |
| message | - | string | - |
| error | - | ErrorInfo | ErrorInfo |
| &nbsp;&nbsp;&nbsp;&nbsp;code | - | string | - |
| &nbsp;&nbsp;&nbsp;&nbsp;message | - | string | - |
| &nbsp;&nbsp;&nbsp;&nbsp;details | - | string | - |
| timestamp | - | string(date-time) | string(date-time) |

### 响应示例

```javascript
{
  "success": true,
  "data": {},
  "message": "",
  "error": {
    "code": "",
    "message": "",
    "details": ""
  },
  "timestamp": ""
}
```

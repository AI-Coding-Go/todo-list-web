## 获取当前提醒

| 项目 | 内容 |
| :--- | :--- |
| **接口地址** | `/api/reminders` |
| **请求方式** | `GET` |
| **请求数据类型** | `application/x-www-form-urlencoded` |
| **响应数据类型** | `*/*` |
| **接口描述** | 获取当前需要提醒的任务列表（未来30分钟内到期） |

### 请求参数

暂无

### 响应状态

| 状态码 | 说明 | schema |
| :--- | :--- | :--- |
| 200 | OK | ApiResponseListTodoTaskResponse |
